# Results folder

This folder contains processed outputs generated by scripts.

## Current files

- `uniprot_O67940_record_summary.txt`
  - Generated by: `scripts/parse_uniprot_O67940.py`
  - Purpose: parsed summary of the full UniProtKB JSON record for O67940_AQUAE.

- `O67940_interpro_pfam_metadata.tsv`
  - Generated by: `scripts/fetch_interpro_pfam_metadata.py`
  - Purpose: summarizes InterPro and Pfam metadata for the cross-references found in the current UniProtKB record of O67940_AQUAE.

- `alphafold/O67940_alphafold_plddt_summary.txt`
  - Generated by: `scripts/summarize_alphafold_plddt.py`
  - Purpose: summarizes per-residue pLDDT confidence values extracted from the AlphaFoldDB v6 PDB model for O67940_AQUAE.

- `pdb/reference_structure_download_summary.tsv`
  - Generated by: `scripts/fetch_reference_pdb_structures.py`
  - Purpose: records file sizes, ATOM counts, and download status for reference structures 2Q6O and 1RQP.

- `pdb/reference_pdb_chain_summary.tsv`
  - Generated by: `scripts/inspect_reference_pdb_contents.py`
  - Purpose: summarizes protein chains, ATOM counts, and approximate residue counts in the reference PDB files.

- `pdb/reference_pdb_heteroatom_summary.tsv`
  - Generated by: `scripts/inspect_reference_pdb_contents.py`
  - Purpose: summarizes HETATM records such as ligands, ions, cofactors, and water molecules in the reference PDB files.

- `pdb/reference_chainA_sequence_summary.tsv`
  - Generated by: `scripts/extract_structure_chain_sequences.py`
  - Purpose: summarizes chain A sequences extracted from the O67940 AlphaFold model and reference structures 2Q6O and 1RQP.

- `structure_comparison/tmalign_pairwise_summary.tsv`
  - Generated by: `scripts/run_tmalign_pairwise.py`
  - Purpose: summarizes pairwise TM-align structural comparisons between O67940_AQUAE AlphaFold chain A, 2Q6O chain A, and 1RQP chain A.

- `structure_comparison/tmalign_raw/`
  - Generated by: `scripts/run_tmalign_pairwise.py`
  - Purpose: stores raw TM-align outputs for each pairwise structural comparison.

- `evidence_integration/evidence_integration_table.tsv`
  - Generated by: `scripts/create_evidence_integration_table.py`
  - Purpose: integrates all major evidence layers and separates observations, supported interpretations, unsupported claims, and limitations.

- `evidence_integration/evidence_integration_table_short.tsv`
  - Generated by: `scripts/create_evidence_integration_table_short.py`
  - Purpose: shorter version of the evidence integration table suitable for report drafting.
